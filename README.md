AI-Powered Career Advisor Chatbot using Gemini GenAI

Project Overview



This project implements a Production-Ready Domain-Specific Career Advisor Chatbot powered by the Google Gemini GenAI API.



The chatbot provides structured, intelligent, and career-focused guidance while maintaining contextual memory across multiple user interactions.



The system follows production-grade AI architecture principles, including modular backend design, secure API handling, structured prompt engineering, logging, and cloud deployment on AWS EC2.



Key Features



Domain-Specific Career Guidance



Multi-Turn Conversation Memory



Structured Prompt Engineering



Modular Backend Architecture



Secure API Key Management using Environment Variables



Logging of API Calls and Errors



Interactive Streamlit Chat UI



Cloud Deployment on AWS EC2 (Public Access)



System Architecture



User

→ Streamlit UI

→ Backend Layer

→ Prompt Engineering Module

→ Gemini API

→ Response Processing

→ UI Rendering



The system ensures separation of concerns by isolating API logic, prompt management, memory handling, and UI components.
